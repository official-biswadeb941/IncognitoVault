This error occurs when the server cannot satisfy the range specified in the `Range` header of the request.

Possible Causes:

  • Invalid Range: The specified range is not valid for the resource.
  • Resource Size: The requested range may exceed the size of the resource.
  • Request Error: The request header might contain incorrect or unsupported range values.

Next Steps:

  • Check Range: Verify that the range specified in the request is within the bounds of the resource.
  • Correct Request: Adjust the request to ensure it aligns with the available ranges of the resource.
  • Contact Support: Reach out to support if you need help understanding or correcting the range values.
